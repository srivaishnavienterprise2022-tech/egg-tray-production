
# Egg Tray Production Pro

ఈ యాప్ ఎగ్ ట్రే తయారీ కంపెనీ కోసం ప్రొడక్షన్ వివరాలను నమోదు చేయడానికి తయారు చేయబడింది.

## ఫీచర్లు:
- **Offline Mode:** ఇంటర్నెట్ లేకపోయినా డేటా సేవ్ అవుతుంది.
- **Machine-wise Tracking:** 16lb మరియు 20lb మెషీన్ల కోసం ప్రత్యేక ఎంట్రీలు.
- **AI Insights:** Gemini AI ద్వారా ప్రొడక్షన్ సారాంశం.
- **PWA Ready:** మొబైల్ హోమ్ స్క్రీన్ మీద యాప్ లాగా ఇన్‌స్టాల్ చేసుకోవచ్చు.

## Deployment:
1. Upload to GitHub.
2. Connect to Vercel.
3. Add `API_KEY` in Vercel Environment Variables.
